These scripts assume you have already run 
```
$ gcloud init 
```
The google-cloud-texttospeech library will look for ADC's generated by gcloud. 
If you don't know where your ADCs are rerun that command. 
It is typically in `$HOME/.config/gcloud/application_default_credentials.json`

Here's a test if you want to confirm the APIs enabled
```
$ python3 -m venv venv 
$ source venv/bin/activate
$ pip3 install google-cloud-texttospeech
$ python3
>>> from google.cloud import texttospeech
>>> testtospeech.TextToSpeechClient().list_voices()
```

Invocation
```
python3 tts.py --input ./ssml/slothbear1.ssml --output ./audio/slothbear1.mp3 --limit 1000000 --count-file wc.txt
```
